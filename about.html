About

Paulo Verde Technologies is an Arizona-based company that provides custom computer programming and technical solutions. We specialize in application and web development, especially with C/C++, Java, and Node.js. For more information about the services we offer, please see pauloverde.com/services.html.

One project we have worked on in the past is the original back-end for the website csgogamble.com, which included Steam API incorporation, database usage, and real-time server-browser communication.

As another example, we have developed the custom search bar and weather forecast features for BestHomepageEver, which included integration with the OpenWeatherMap API.

We are always looking to take on new challenges and work on interesting projects. Please email us at contact@pauloverde.com for assistance developing or extending a project of your own.
